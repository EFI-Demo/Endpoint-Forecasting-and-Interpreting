
		Today Microsoft released the latest Security Intelligence
Report (SIR). The ninth volume of this
comprehensive report covers evolving threat landscape trends from January
through June 2010, with a focus on botnets.
You can read the
report section by section online,
or you can download
the entire report.
Here are some of the key findings:


Older
operating systems are more vulnerable to infections than newer versions.

Most
attackers use social engineering techniques to trick you into installing
malware. 

Stolen
equipment remains the most frequent type of security breach incident.

Windows Update and Microsoft Update
usage has increased 75 % since the second half of 2006.
